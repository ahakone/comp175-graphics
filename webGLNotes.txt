--- Notes from 4.28.16 --- 
- Attribute (only accessible to vertex shader bc per vertex level)
  - vec3 position
  - vec3 normal
  - vec3 color
  - vec2 texcoord
     
- Uniform (accessible to both vertex and fragment shader)
- Varying 
  - Propagate information between vertex to fragment
  - Interpolates (you can speicify linear or non-linear) from per vertex to per pixel
    - e.g., Interpolates normals of a point using the vertices
  - e.g., 
  vec3 normal // attribute
  varying vec3 vnormal;
  main () {
       vnormal = normal;
  }
